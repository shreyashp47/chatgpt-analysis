# ChatGPT Clone

A ChatGPT clone built with Next.js, TypeScript, Tailwind CSS, and a provider-agnostic AI layer supporting Groq, Gemini, OpenAI, and more.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **AI Providers:** Groq (default), Google Gemini, OpenAI, Mistral, etc.
- **State:** Zustand + LocalStorage
- **Markdown:** react-markdown + rehype-highlight
